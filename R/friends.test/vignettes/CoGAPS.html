Genes that are friends to CoGAPS patterns (or CoGAPS patterns that are friends to genes?)

# get CoGAPS result from zenodo
fname <- 'cogapsresult.Rds'
if(!file.exists(fname)){
    download.file('https://zenodo.org/records/7709664/files/cogapsresult.Rds', destfile = fname, mode = 'wb')
}
cg <- readRDS('cogapsresult.Rds')

# get feature loadings
loadings <- cg@featureLoadings

Let’s find representative genes all patterns using friends.test. We are interested in all gene-friends for all patterns, so patterns need to be in rows, and genes in columns.

friends_genes <- friends.test::friends.test(t(loadings))

What happens if we don’t transpose?

friends_patterns <- friends.test::friends.test(loadings, max.friends.n = 1)

qq <- lapply(friends_patterns, function(x) names(x)) |> as.data.frame() |> t()

by_patterns <- lapply(unique(qq), function(pat) rownames(qq)[qq==pat])

names(by_patterns) <- unique(qq)
